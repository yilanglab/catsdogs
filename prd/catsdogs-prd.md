# 猫人狗人 - 三维动物人格测试游戏 PRD

> 版本：v1.0 | 日期：2026-04-10

---

## 一、产品概述

### 1.1 一句话定义

一个通过 15 道场景化情境题，从「能量」「策略」「内核」三个维度定位你的隐藏动物人格的趣味测试游戏。

### 1.2 核心卖点

- **直觉共鸣**：每个人都能秒懂"你是猫人还是狗人"
- **深度有层次**：不止猫/狗二分，而是 18 种具体品种，精准到"哪种猫、哪种狗"
- **社交驱动**：自测 vs 他测的认知差异、CP 配对、团队图鉴，天然产出社交话题
- **可传播**：一句话人设 + 三标签 + 扎心语录，适合截图分享

### 1.3 目标用户

- 18-35 岁，对性格测试、MBTI 类内容有兴趣的年轻人
- 社交场景：朋友聚会破冰、情侣互测、办公室团建

## 二、体系模型

### 2.1 三维洋葱模型

由外到内三层，每层 3 个档位（内核层 2 个），组合出 3 × 3 × 2 = **18 种人格品种**。

#### 外层：能量维度 (Energy)

观察一个人的活跃度与耗能模式。

| 档位 | 代号 | 关键词 | 描述 |
|------|------|--------|------|
| 低能量 | E-Low | 安静/内收/独处 | 倾向于独处充电，在人群中存在感低，情绪内收 |
| 中能量 | E-Mid | 温和/流动/适度 | 可静可动，情绪稳定，社交表现得体且松弛 |
| 高能量 | E-High | 活跃/外放/焦点 | 精力旺盛，情绪写在脸上，是环境中的能量发射源 |

#### 中层：策略维度 (Strategy)

观察一个人处理冲突、利益及人际关系的手段。

| 档位 | 代号 | 关键词 | 描述 |
|------|------|--------|------|
| 纯系 | S-Pure | 钝感/信赖/无害 | 无防备，防御机制简单，真诚且无害 |
| 狐系 | S-Fox | 机敏/适度/高情商 | 高社会化程度，懂得审时度势，灵活底线，擅长规避风险 |
| 狼系 | S-Wolf | 审视/掌控/领地 | 极强目标感和领地意识，带攻击性或压迫感，倾向掌控局面 |

#### 内核：驱动维度 (Core)

观察一个人灵魂深处最本质的需求。

| 档位 | 代号 | 关键词 | 描述 |
|------|------|--------|------|
| 猫系 | C-Cat | 自我驱动/边界/独立 | 关注"我舒不舒服"、"逻辑是否自洽"，追求独立与自由 |
| 犬系 | C-Dog | 关系驱动/认可/归属 | 关注"是否被认可"、"是否被连接"，追求归属感与反馈 |

## 三、问卷设计

### 3.1 设计原则

1. **场景化**：每道题是一个具体情境而非抽象评价，降低社会期望效应
2. **隐藏维度**：答题者无法直接看出选项对应什么倾向
3. **题目打乱**：三个维度的题目交叉排列，不按维度分组
4. **总量 15 题**：每个维度 5 题，3-5 分钟完成，符合轻量级测试定位

### 3.2 计分规则

#### 能量维度（5 题）

每题选项隐含分值：
- 低能量选项 = 1 分
- 中能量选项 = 2 分
- 高能量选项 = 3 分

总分 5-15 分：
- **5-8 分** → 低能量 (E-Low)
- **9-11 分** → 中能量 (E-Mid)
- **12-15 分** → 高能量 (E-High)

#### 策略维度（5 题）

每题选项隐含分值：
- 纯系选项 = 1 分
- 狐系选项 = 2 分
- 狼系选项 = 3 分

总分 5-15 分：
- **5-8 分** → 纯系 (S-Pure)
- **9-11 分** → 狐系 (S-Fox)
- **12-15 分** → 狼系 (S-Wolf)

#### 内核维度（5 题）

每题选项隐含分值：
- 猫系选项 = 1 分
- 犬系选项 = 2 分

总分 5-10 分：
- **5-7 分** → 猫系内核 (C-Cat)
- **8-10 分** → 犬系内核 (C-Dog)

> 注：7 分为临界值。当恰好为 7 分时，偏猫但带有犬系色彩，结果描述中会额外注明"你身上带有一定的犬系特质"。反之 8 分偏犬也类似。5-6 分为强猫，9-10 分为强犬。

### 3.3 题目内容

> 以下题目中，三个维度交叉排列。选项顺序在实际产品中应随机打乱（不总是 ABC 对应同一方向）。

周五晚上，朋友突然拉了个 10 人群说今晚聚餐，你的第一反应是——

- A. 假装没看到消息，默默祈祷别人不要 @ 自己 → 1分
- B. 回一个"好呀～几点？"然后继续手头的事 → 2分
- C. 秒回"终于！我来选餐厅！"并开始疯狂 @ 所有人 → 3分

## 五、社交玩法

> **设计原则**：Phase 1-2 的所有社交功能均通过 URL 参数传递数据，不需要后端、不需要账号、不存储任何用户个人信息。

### 5.1 照镜子模式

**逻辑**：同一道题出两套版本——"自测版"（以自己为对象回答）和"他测版"（以朋友为对象回答）。

**流程**：
1. 用户 A 完成自测，得到结果 X
2. 点击"邀请朋友测测我"，自动生成一个他测链接（A 的得分编码在 URL 参数中）
3. 朋友 B 打开链接，进入他测模式，以 A 为对象答题，得到结果 Y
4. B 答完后，前端直接对比 URL 中 A 的自测结果和 B 的答题结果，当场生成「认知差异报告」：
   - 如果 X = Y："你的自我认知很准确，表里如一"
   - 如果 X ≠ Y：分析哪个维度产生了偏差，并给出解读（类似"柯基身布偶皮"的分析）

**差异解读模板**：

> **你眼中的自己：[品种A]**
> **朋友眼中的你：[品种B]**
>
> 你们在 [能量/策略/内核] 维度上出现了偏差。
> 这意味着 [具体解读]。
> 你的真实画像可能是：外表是 [品种B的外层特质]，内心是 [品种A的内核特质]。

### 5.2 CP 配对模式

**逻辑**：A 测完后生成配对链接（A 的得分编码在 URL 中），B 打开链接完成自己的测试后，前端直接拿到双方得分，当场生成关系分析卡。

**内容包含**：
- 双方品种卡片并排展示
- 契合度评分（基于品种关系图谱）
- 关系解读："你们是[xxx]型组合"
- 相处建议：一句话给双方的提醒

**组合类型示例**：
- 猫+猫 = "平行宇宙"型：互不打扰的高级感，但要小心冷到冻住
- 猫+狗 = "互补充电"型：一个给空间一个给温暖，经典搭配
- 狗+狗 = "热情碰撞"型：感情浓度超标，但要注意谁来当理性那个
- 狼+纯 = "保护者联盟"型：一个强一个软，前提是强的那个别太控制

### 5.3 团队图鉴（Phase 3，需轻量后端）

**逻辑**：创建一个临时房间（6 位房间码），成员各自完成测试后提交品种结果到房间，前端汇总渲染团队图鉴。仅存品种编号，不存个人信息，数据 7 天自动过期。

**展示内容**：
- 团队成员的品种分布图
- 团队性格光谱：偏猫还是偏狗？能量中心在哪？
- "你们团队缺一个 [品种]"的趣味提示
- 可下载的团队海报

## 七、技术实现方案

### 7.1 核心原则：纯前端、零后端、不存储用户信息

整个 MVP（Phase 1 + Phase 2）设计为**纯静态网页**，不需要后端服务、不需要用户注册/登录、不存储任何个人信息。所有数据通过 URL 参数在用户之间传递。

### 7.2 数据编码方案

三个维度的得分编码进 URL，作为所有社交玩法的数据载体：

```
https://catsdogs.com/result?e=7&s=10&c=4
```

| 参数 | 含义 | 范围 |
|------|------|------|
| `e` | 能量维度得分 | 5-15 |
| `s` | 策略维度得分 | 5-15 |
| `c` | 内核维度得分 | 5-10 |

- 任何人打开这个链接，前端直接根据三个参数计算品种并渲染结果页
- 链接本身不包含姓名、头像等任何个人信息

### 7.3 各功能的实现方式

#### 核心测试（Phase 1）

纯前端完成：
- 单页应用（SPA），15 道题用滑动卡片式交互
- 每题选项随机排列顺序（防止答题者发现规律）
- 答完后前端计算三维得分，跳转到结果页
- 结果页支持生成分享长图（html2canvas / dom-to-image）
- 分享链接即结果页 URL（含得分参数）

#### 照镜子模式（Phase 2）

纯 URL 参数传递，不需要后端：

**流程：**
1. 用户 A 完成自测，得到结果，例如 `e=9&s=10&c=8`（柯基）
2. 点击"邀请朋友测测我"，生成一个"他测链接"：
   ```
   https://catsdogs.com/mirror?from=e9s10c8
   ```
   `from` 参数是 A 的自测结果编码（紧凑格式：`e{值}s{值}c{值}`）
3. 朋友 B 打开链接，进入"他测模式"（题目措辞切换为"TA 会怎样？"视角）
4. B 答完后，前端同时拿到：
   - A 的自测得分（从 URL `from` 参数解码）
   - B 以 A 为对象的测试得分（刚刚作答的结果）
5. 直接在前端生成「认知差异报告」：对比两组得分，指出哪个维度有偏差

**差异报告逻辑：**
- 三个维度逐一对比，如果落入不同档位则标记为"偏差"
- 偏差维度越多，报告越丰富
- 零偏差 = "表里如一"的特殊结果

#### CP 配对模式（Phase 2）

同样纯 URL 传递：

1. 用户 A 测完后点击"邀请 TA 一起测"，生成配对链接：
   ```
   https://catsdogs.com/pair?a=e9s10c8
   ```
2. 用户 B 打开链接完成自己的测试
3. B 答完后前端拿到双方得分，直接生成 CP 配对分析卡

#### 团队图鉴（Phase 3 · 需要轻量后端）

这是唯一需要后端的功能（多人 > 2 的数据汇聚），可延后实现：
- 使用 Firebase Realtime Database / Supabase 等 BaaS 服务
- 创建临时房间（6 位房间码），成员提交品种结果（仅存品种编号，不存个人信息）
- 数据 7 天自动过期
- 前端根据房间内所有结果渲染团队图鉴

### 7.4 前端技术选型建议

| 方面 | 建议 |
|------|------|
| 框架 | Next.js / Astro / 纯 Vite + React 均可；如追求极致轻量可用纯 HTML + Vanilla JS |
| 部署 | Vercel / Netlify / Cloudflare Pages（免费静态托管） |
| 分享图生成 | html2canvas 或 dom-to-image（前端截图生成 PNG） |
| 动画/交互 | 答题卡片滑动可用 Framer Motion / Swiper |
| 国际化 | 初版仅中文，预留 i18n 结构 |

### 7.5 分期交付建议

**Phase 1 - 核心体验**（纯前端，可 1-2 周完成）：
- 15 题问卷 + 计分逻辑
- 18 种结果页（含品种卡片 + 深度解析 + 拍档/天敌）
- 结果页分享图生成
- 分享链接（URL 带得分参数）

**Phase 2 - 社交裂变**（纯前端，+1 周）：
- 照镜子模式（自测 + 他测 + 差异报告）
- CP 配对模式
- 他测版题目措辞适配

**Phase 3 - 扩展玩法**（需轻量 BaaS，按需）：
- 团队图鉴（临时房间 + 匿名结果汇聚）
- 品种关系图谱可视化探索
- 可选：匿名统计看板（各品种分布比例等趣味数据）
