# 猫人狗人 v2 优化 PRD

> 版本：v2.0 | 日期：2026-04-13
> 基于 v1.0 完整实现的优化迭代

---

## 一、优化总览

### 1.1 优化目标

将结果页从「信息卡片堆叠」升级为「沉浸式深度人格画像」，让用户读完后有强烈的「被说中了」感受。同时精简社交功能、修复 UI 问题、提升整体视觉品质。

### 1.2 优化范围

| 模块 | 动作 | 优先级 |
|------|------|--------|
| 结果页内容扩充 | 新增 4 个深度人格板块（18品种 × 4板块 = 72段文案） | P0 |
| 结果页 UI 重构 | 适配新板块、修复裁切问题、优化视觉节奏 | P0 |
| CP 配对功能 | 完整移除 | P0 |
| 全局 UI 优化 | 整体视觉品质提升 | P1 |

### 1.3 不变的部分

- ✅ 15 道题问卷系统（题目、计分、判定逻辑不变）
- ✅ 18 种品种体系（三维模型不变）
- ✅ 照镜子功能（保留不动）
- ✅ 分享功能（链接 + 分享图）
- ✅ 首页、答题页（功能不变，仅 UI 优化）

## 二、结果页内容扩充（重点）

### 2.1 设计原则

1. **延续 PRD v1 调性**：短句、画面感、略带扎心、不说教
2. **场景化 > 抽象化**：用具体场景描述而非性格标签堆砌
3. **差异化**：18 个品种的同一板块内容不能雷同，要体现品种间的本质差异
4. **可共鸣**：读者看完应该想截图发朋友圈说「太准了」
5. **控制篇幅**：每个子字段 1-2 句话，整个结果页阅读时间约 5 分钟

### 2.2 新增数据结构

在 `Breed` 类型中新增以下字段：

```typescript
interface Breed {
  // ... 现有字段保持不变 ...

  /** 🧠 内在机制 */
  innerMechanism: {
    /** 压力反应：压力来临时的本能反应模式 */
    stressResponse: string;
    /** 充电方式：恢复能量的方式 */
    rechargeMode: string;
    /** 情绪模式：日常情绪的运作模式 */
    emotionalPattern: string;
  };

  /** 💼 职场风格 */
  workStyle: {
    /** 当领导时的管理风格 */
    asLeader: string;
    /** 当下属时的工作态度 */
    asFollower: string;
    /** 开会时的表现 */
    meetingStyle: string;
  };

  /** ❤️ 恋爱模式 */
  loveStyle: {
    /** 心动信号：喜欢一个人的表现 */
    crushSignal: string;
    /** 雷点：恋爱中的绝对禁忌 */
    dealBreaker: string;
    /** 理想相处：期望的亲密关系状态 */
    idealRelationship: string;
  };

  /** 🤝 社交习惯 */
  socialHabit: {
    /** 在朋友群里的角色定位 */
    friendGroupRole: string;
    /** 社交电量描述 */
    socialBattery: string;
    /** 友谊中的雷区 */
    friendshipRedFlag: string;
  };
}
```

### 2.3 内容创作规范

#### 文案风格指南

- **人称**：统一用「你」，直接对话感
- **句式**：短句为主，不超过 50 字/句。可以用破折号、省略号增加节奏
- **语气**：像一个很了解你的朋友在跟你聊天，不是心理咨询师在分析你
- **禁忌**：
  - ❌ 不用「可能」「也许」「大概」等模糊词（要斩钉截铁才有「说中了」的感觉）
  - ❌ 不堆砌心理学术语
  - ❌ 不做正面教育/成长建议（这不是鸡汤）
  - ❌ 不同品种的同一字段不能出现相似表达

#### 内容差异化矩阵

每个板块的内容应与该品种的三维定位强关联：

| 维度组合 | 内在机制倾向 | 职场风格倾向 | 恋爱模式倾向 | 社交习惯倾向 |
|---------|-------------|-------------|-------------|-------------|
| 低能量+纯系+猫 | 回避/内收 | 独立/低存在 | 被动/高门槛 | 隐形/低频 |
| 高能量+狼系+犬 | 爆发/外化 | 主导/高强度 | 主动/高投入 | 中心/高频 |
| 中能量+狐系+猫 | 理性拆解 | 灵活/精算 | 策略/留余地 | 精准/高效 |
| ... | 以此类推 | | | |

### 2.4 全部 18 品种内容模板

以下为每个品种的 4 板块内容。**文案创作时必须以 PRD v1 中的品种描述为基础人设，确保新板块与「一句话人设」「超能力」「软肋」「扎心一句」的人格调性一致。**

#### 🐱 猫系内核 × 纯系策略

##### 英国短毛猫 | 佛系禅师

**🧠 内在机制**
- 压力反应：自动进入「世界与我无关」模式，等压力自己过去。如果过不去——那就适应它。
- 充电方式：一个人窝着，什么也不做也不想，「发呆」就是你最高效的充电方式。
- 情绪模式：情绪波动极小，不是没有情绪，是早就跟情绪达成了和平协议。

**💼 职场风格**
- 当领导时：佛系放权派，只要结果OK过程随意。下属爱你，但你的上司可能觉得你缺乏野心。
- 当下属时：最不需要被管的员工。给我任务、给我deadline、然后别来烦我。
- 开会风格：全程安静，只在被点名时用最少的字回答最关键的问题。

**❤️ 恋爱模式**
- 心动信号：开始愿意为对方打破自己的日常节奏——这对你来说已经是惊天动地了。
- 雷点：对方试图「改造」你，或者觉得你的生活方式是「不上进」。
- 理想相处：各自躺着，偶尔交换一个眼神，不说话也很舒服。

**🤝 社交习惯**
- 朋友群角色：潜水冠军，每年冒泡三次，但每次出现大家都很惊喜。
- 社交电量：出厂就是省电模式，聚会超过 90 分钟开始心不在焉。
- 友谊雷区：不请自来的热情、强行组局、以及「你怎么又不出门」的关心。

##### 美国短毛猫 | 随和室友

**🧠 内在机制**
- 压力反应：不会崩溃也不会爆发，就是默默消化，消化完了跟没事一样。
- 充电方式：做点简单的、不需要动脑的事——刷剧、散步、整理房间。
- 情绪模式：情绪曲线几乎是一条直线，偶尔有波动也能迅速回归均值。

**💼 职场风格**
- 当领导时：不会给团队压力，也不会画大饼。稳稳地推进，不出彩但也不出错。
- 当下属时：让干啥干啥，不抱怨不邀功，是领导最省心的存在。
- 开会风格：不抢话、不反对、偶尔点头。散会后默默把该做的做了。

**❤️ 恋爱模式**
- 心动信号：对你的回复速度从「随缘」变成了「秒回」——虽然内容还是很淡定。
- 雷点：戏太多、情绪波动大、动不动就要你表态站队。
- 理想相处：不需要轰轰烈烈，就是两个人在一起比一个人舒服一点点就够了。

**🤝 社交习惯**
- 朋友群角色：被遗忘也不生气的那个人，但大家想起来都觉得「对哦还有 TA」。
- 社交电量：中等偏低，能社交但绝不主动发起，属于「你来我就接」型。
- 友谊雷区：被迫当传话筒、被卷入别人的纷争、需要你选边站的场合。

##### 奶牛猫 | 神经质乐子人

**🧠 内在机制**
- 压力反应：先疯一阵，然后突然冷静下来，用一种完全出人意料的角度解决问题。
- 充电方式：做一件从没做过的事——哪怕只是换条路回家。新鲜感就是你的充电宝。
- 情绪模式：情绪像过山车但复位速度极快，上一秒崩溃下一秒已经在笑了。

**💼 职场风格**
- 当领导时：团队里最有创意的领导，但下属永远猜不到你下一步要干什么。
- 当下属时：常规任务做得一般，但一旦遇到「别人搞不定的怪题」就是你的主场。
- 开会风格：要么走神到外太空，要么突然说出一个天才想法让所有人愣住。

**❤️ 恋爱模式**
- 心动信号：开始对对方进行「密集的无厘头骚扰」——恭喜你，这就是表白。
- 雷点：无聊。对方一旦变得可预测，你就开始想找新的刺激了。
- 理想相处：每天都有新鲜事，两个人在一起比单口相声还好笑。

**🤝 社交习惯**
- 朋友群角色：气氛组组长，负责在群死的时候扔出一个离谱的话题。
- 社交电量：取决于有没有好玩的事。有趣的局能续到凌晨，无聊的局 10 分钟走人。
- 友谊雷区：太正经、太无聊、以及试图用「成熟」来压制你的快乐。

#### 🐱 猫系内核 × 狐系策略

##### 俄罗斯蓝猫 | 清冷智性恋

**🧠 内在机制**
- 压力反应：把自己关进精神小黑屋，用逻辑把问题拆解到没有情绪的程度。如果问题无法解决，就说服自己不在乎。
- 充电方式：一个人待着、不被打扰、做些不需要回应任何人的事。安静到能听见自己思考的声音。
- 情绪模式：外表永远平静如水，但内心有一套精密的情绪仪表盘，只不过只有极少数人能读懂。

**💼 职场风格**
- 当领导时：不会频繁开会刷存在感，但你发布的每条指令都逻辑严密、滴水不漏。下属会觉得你冷，但不得不服。
- 当下属时：理想上司是「给我目标然后别管我」型。最受不了事无巨细的微管理，会在内心默默给领导扣分。
- 开会风格：能不开就不开，非开不可就用最少的话把最关键的信息说完。

**❤️ 恋爱模式**
- 心动信号：开始主动找你说话了——不是闲聊，而是分享那些平时不会告诉别人的想法。
- 雷点：逻辑混乱、过度黏人、没有自己的精神世界。以及——在你独处时不请自来。
- 理想相处：各自有独立空间，但偶尔对视时知道「这个人懂我」。不需要秀恩爱，但需要灵魂共振。

**🤝 社交习惯**
- 朋友群角色：那个很少发言但每次说话都被截图的人。
- 社交电量：出厂自带省电模式，聚会超过 2 小时自动进入飞行模式。
- 友谊雷区：不尊重边界的人、自来熟、毫无铺垫就要你分享隐私的人。

##### 布偶猫 | 优雅利己者

**🧠 内在机制**
- 压力反应：表面云淡风轻，内心已经把 Plan B 到 Plan D 全想好了。绝对不会让你看到手忙脚乱的样子。
- 充电方式：精致的独处——泡澡、听音乐、整理空间。充电环境必须是「有质感的舒适」，绝不是随便躺着。
- 情绪模式：情绪波动有，但从不外露。你的微笑是标准配置，不代表心情。

**💼 职场风格**
- 当领导时：温柔但有距离感的管理者，擅长让下属觉得「被尊重」的同时把活推出去。
- 当下属时：表面配合度极高，但内心有一把清晰的算盘：这件事对我有什么好处。
- 开会风格：发言永远得体、不得罪人，但仔细复盘你会发现——你没有承诺任何事。

**❤️ 恋爱模式**
- 心动信号：愿意在你面前卸下「得体」的伪装，露出真实的、不那么完美的一面。
- 雷点：粗糙、不体面、在公共场合让你丢脸。以及——试图强行闯入你精心维护的秩序。
- 理想相处：外人看来是令人羡慕的优雅情侣，两人独处时可以不端着、不演。

**🤝 社交习惯**
- 朋友群角色：看起来跟谁都好，但深交名单上一共就三五个人。
- 社交电量：能社交很久但会精确计算投入产出比，无效社交一秒都不浪费。
- 友谊雷区：突破你精心维护的社交距离、把你的客气当真心、在背后说你「虚伪」。

##### 阿比西尼亚猫 | 聪明探险家

**🧠 内在机制**
- 压力反应：立刻开始找新的出路。压力对你来说不是墙，是一扇还没打开的门。
- 充电方式：学一个新技能、去一个没去过的地方、跟一个有趣的陌生人聊天——新鲜感就是你的命。
- 情绪模式：情绪来得快去得也快，不会在同一个情绪里待太久，因为下一个好奇心已经到了。

**💼 职场风格**
- 当领导时：最擅长 0 到 1 的创业型领导，一旦进入稳定运营期就开始坐不住了。
- 当下属时：常规工作让你痛苦，但如果任务有「从没有人做过」的成分，你能爆发 200% 的产出。
- 开会风格：别人在讨论 A 方案和 B 方案时，你已经在想 C 方案了。

**❤️ 恋爱模式**
- 心动信号：开始把对方纳入自己的探险版图——带你去我发现的宝藏小店、分享我最新的兴趣。
- 雷点：试图把你关在笼子里。控制你的时间表、限制你的交友圈、或者让你只能做「一件事」。
- 理想相处：两个人一起探索世界，各自也有各自的冒险。在一起永远有新话题聊。

**🤝 社交习惯**
- 朋友群角色：那个带大家去新地方、安利新事物的人。朋友圈更新频率最高。
- 社交电量：超大容量，但只给「有趣」的人充值。无聊的社交比独处还累。
- 友谊雷区：一成不变的人、只会抱怨但不行动的人、以及想让你「稳定下来」的人。

#### 🐱 猫系内核 × 狼系策略

##### 缅因猫 | 威严领主

**🧠 内在机制**
- 压力反应：更加沉默，但内心已经开始部署方案。你处理压力的方式是「掌控它」，而不是逃避或发泄。
- 充电方式：在自己的领地里，按照自己的节奏，做自己说了算的事。核心需求是「一切尽在掌控」。
- 情绪模式：深水区。表面波澜不惊，底下暗流涌动，但你绝不会让别人看到。

**💼 职场风格**
- 当领导时：天生的掌权者，不怒自威。你不需要拍桌子，一个眼神就够了。
- 当下属时：只服比你强的人。如果领导的水平不如你，你会非常痛苦——但依然沉默。
- 开会风格：通常最后发言，一锤定音。你说完之后，没有人会再质疑。

**❤️ 恋爱模式**
- 心动信号：对你展现出「保护欲」——不是嘴上说，而是默默替你挡掉麻烦。
- 雷点：挑战你的权威、在外人面前不给你面子、试图打破你的规则。
- 理想相处：你掌舵，对方安心做乘客。但你选择的乘客，必须是你真正尊重的人。

**🤝 社交习惯**
- 朋友群角色：定海神针，平时不响，关键时刻一句话稳住全场。
- 社交电量：不高，但在自己的圈子里可以很放松。核心是「我选择跟谁社交」。
- 友谊雷区：不守规矩的人、挑战你底线的人、以及——在你的领地上放肆的人。

##### 暹罗猫 | 强势管理者

**🧠 内在机制**
- 压力反应：嘴先行动。压力一来就开始分析、吐槽、甚至怼人，说完之后反而冷静了。
- 充电方式：跟信任的人倾诉——不是要建议，是要一个听你输出的人。倾诉完就满血了。
- 情绪模式：情绪外化且密度高，开心恨不得全世界知道，生气恨不得全世界闭嘴。

**💼 职场风格**
- 当领导时：高效但高压的管理者。团队被你推着走，效率极高，但离职率可能也不低。
- 当下属时：会公开质疑不合理的决策。你的「提意见」经常被当成「找茬」。
- 开会风格：主导发言权，会打断跑题的讨论，以及直接指出「这个方案不行」。

**❤️ 恋爱模式**
- 心动信号：开始管你的生活——你吃了吗、穿这么少不冷吗、那个朋友我觉得不靠谱。
- 雷点：不回消息、敷衍你的感受、以及——你觉得对方在跟你「装」。
- 理想相处：吵吵闹闹但谁也离不开谁，你管 TA、TA 让着你，偶尔反驳让你觉得有趣。

**🤝 社交习惯**
- 朋友群角色：群里的意见领袖，重大决定（吃什么、去哪玩）都等你拍板。
- 社交电量：极高，但都花在「输出」上了。你是给别人充电的那个人。
- 友谊雷区：阳奉阴违、当面不说背后乱说、以及——你觉得对方在「敷衍」你。

##### 孟加拉豹猫 | 野心征服者

**🧠 内在机制**
- 压力反应：压力是燃料。越有压力越兴奋，没有挑战的日子对你来说才是真正的痛苦。
- 充电方式：做高强度的事——运动、竞技、或者任何能让你感觉「在赢」的活动。
- 情绪模式：情绪永远在高位运行。你不太有「平淡」的时候，要么在巅峰要么在低谷。

**💼 职场风格**
- 当领导时：狼性领导，目标清晰、执行凶猛。团队跟你干很累但成长很快。
- 当下属时：你不会永远当下属。一旦觉得天花板到了，要么晋升要么走人。
- 开会风格：直奔结果，不聊过程。你的口头禅是「所以结论是什么？」

**❤️ 恋爱模式**
- 心动信号：把对方当成「征服目标」——不是物化，而是你用追事业的劲头去追人。
- 雷点：对方太弱、太无趣、或者太容易得到。没有挑战性的关系让你窒息。
- 理想相处：势均力敌的对手型恋人，你们在一起是因为彼此都很强，而不是因为谁需要谁。

## 五、全局 UI 优化（impeccable）

### 5.1 优化方向

使用 impeccable 技能进行以下维度的优化：

1. **Audit** — 跑一遍全局质量检查（可访问性、性能、响应式）
2. **Polish** — 修复对齐、间距、一致性等微细节
3. **Arrange** — 优化新增板块后的整体布局节奏
4. **Typeset** — 检查长内容页面的排版层次
5. **Adapt** — 确保所有页面在移动端微信浏览器中体验良好

### 5.2 关键约束

- 移动端优先（微信内置浏览器为主要环境）
- 猫系冷色 / 犬系暖色的双主题一致性
- 品种专属色的合理运用
- 新板块（内容大量增加后）的阅读舒适度
